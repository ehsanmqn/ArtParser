## Getting started

#### Clone the repository

```bash
git clone https://github.com/ehsanmqn/ArtParser
```

#### Prepare and run
```bash
cd ArtParser
virtualenv -p python3 venv
source venv/bin/activate
python parser.py
```

#### Happy coding!
